Bu proje, modern web geliştirmeye giriş yapmak amacıyla HTML ve CSS kullanılarak hazırlanmış basit, temiz ve responsive bir web sayfasıdır. Proje; sayfa yapısı, stil düzenlemeleri, görsel yerleşimleri ve mobil uyumlu tasarım gibi temel front-end pratiklerini uygulamak için geliştirilmiştir.

---

## Kullanılan Teknolojiler

- HTML5  
- CSS3  
- Responsive Design (tüm sayfa yapısı)  
- Flexbox  
- Vercel (Deployment)

---

## Özellikler

- Temiz ve düzenli HTML yapısı  
- Harici CSS ile global stil yönetimi  
- Mobil uyumlu (responsive) sayfa tasarımı  
- Görseller için düzenli klasör yapısı  
- Basit ve anlaşılır kullanıcı arayüzü  
- Vercel üzerinde hızlı deploy
